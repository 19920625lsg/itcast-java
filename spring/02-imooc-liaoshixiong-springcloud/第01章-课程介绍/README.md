## 本课程重点介绍的内容

![本课程重点介绍的内容](本课程重点介绍的内容.jpg)

+ Eureka
  + Eureka Server
  + Eureka Client
  + Eureka 高可用
  + 服务发现机制
+ Config
  + Config Server
  + Config Client
  + Spring Cloud Bus(结合RabbitMQ)自动刷新
+ Ribbon
  + RestTemplate
  + Feign
  + Ribbon源码分析，了解底层
+ Zuul
  + 动态路由
  + 校验
+ Hystrix
  + Hystrix Dashboard
  + 熔断机制
+ 容器编排和服务追踪
  + 容器编排: Docker + Rancher
  + 服务追踪：Spring Cloud Sleuth + Zipkin
