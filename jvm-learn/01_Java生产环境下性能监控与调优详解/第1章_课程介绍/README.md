# 课程介绍

## 本课程帮你迈过哪些坎？

+ 1、生产环境发生了内存溢出该如何处理？
+ 2、生产环境应该给服务器分配多少内存合适？
+ 3、如何对垃圾收集器的性能进行调优？
+ 4、生产环境CPU负载飙高该如何处理？
+ 5、生产环境应该给应用分配多少线程合适？
+ 6、不加log如何确定请求是否执行了某一行代码？
+ 7、不加log如何实时查看某个方法的入参与返回值？
+ 8、JVM的字节码是什么东西？
+ 9、循环体中做字符串+拼接为什么效率低？
+ 10、字符串+凭借一定就是StringBuilder.append吗？
+ 11、String常量池是怎么回事？
+ 12、i++与++i到底哪种写法效率更高？

## 本课程收获

+ 1、熟练使用各种监控和调试工具
+ 2、从容应对生产环境中遇到的各种调试和性能问题
+ 3、熟悉JVM字节码指令
+ 4、深入理解JVM的自动回收机制，学会GC调优
